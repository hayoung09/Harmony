  • HTML(HYPERTEXT MARKUP LANGUAGE)

    What was html?

    HTML is not a programming language; it is a markup language that defines the structure of content. HTML consists of a series of **elements** that wrap and wrap different parts of the content to make it appear in different formats or behave in a particular way. You can hyperlink words or images to somewhere else, italicize words and make your font bigger or smaller by wrapping them in tags.

  • JAVASCRIPT

    What was javasript?

    JavaScript (JS) is a lightweight object-oriented interpreter language that uses first-class functions and is well known as the scripting language of web pages, but is also widely used in non-browser environments. It is a prototype-based, multi-paradigm script language and supports dynamic instruction, object-oriented, and functional programming styles. JavaScript runs on the client-side web (browser) and allows you to design/program how a web page works when an event. JavaScript is widely used to control web page behavior in an easy-to-learn and powerful scripting language.
